# 2D Ising Model Simulation (Metropolis Algorithm)

This project implements a 2D Ising model using the Metropolis Monte Carlo method to study phase transitions.

## Features
- Simulation of spin lattice
- Magnetization vs Temperature
- Energy vs Temperature
- Specific Heat vs Temperature
- Susceptibility vs Temperature
